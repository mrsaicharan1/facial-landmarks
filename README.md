# facial-landmarks
A module used to detect faces and 68 facial landmarks(Eyes, nose, chin etc.) in a given image.
This module can be further utilised for face recongition systems.

# Instructions

The shape detector file which is to be included for facial landmark detection can be found http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2

# How to run
`python detect.py --shape-detector <shape detector file> --image <image_file>`
